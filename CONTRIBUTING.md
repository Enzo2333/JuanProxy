# Contributing

Thanks for helping improve JuanProxy.

## Development Setup

Requirements:

- Node.js 22 or newer
- npm

Install dependencies:

```powershell
npm install
```

Run the app:

```powershell
npm start
```

Run tests:

```powershell
npm test
```

## Pull Request Checklist

- Keep real API keys, passwords, cookies, and private dashboard URLs out of commits.
- Use `example.com` domains for fixtures, docs, and screenshots.
- Run `npm test` before opening a pull request.
- Update `README.md` or `CHANGELOG.md` when user-facing behavior changes.

## Project Notes

JuanProxy is an Electron desktop app for managing OpenAI-compatible upstream API endpoints and forwarding local proxy requests. The app intentionally keeps user configuration local.
